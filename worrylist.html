What is this site?

Help for Various Mental Woes is designed to help you help yourself when you are feeling bad.

If you are experiencing anxiety, you could try self-soothing.

If you are experiencing the urge to do an addictive behavior, you could try urge management.

If you are experiencing painful emotions, you could read about emotions.

If you need something to do, you could look for a distraction.

Perhaps you'd like to see our famous apology flowchart.

Worry List

Write a list of all the things that you can think of that you are worried about. However, set a timer to 2-5 minutes and only write during that time. After the time is done, try to stop dwelling on the worries.
